# Task Flow 🚀

Task Flow is a modern, premium TODO application built with **Django 5.1**. It features a stunning glassmorphism design, interactive task management, and a clean, responsive interface.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.10%2B-green)
![Django](https://img.shields.io/badge/django-5.1-red)

## ✨ Features

- **Premium UI**: Modern glassmorphism aesthetic with smooth gradients and animations.
- **Task Management**: Create, edit, delete, and toggle task completion.
- **Interactive UI**: Real-time feedback and sleek task cards.
- **Responsive**: Works perfectly on mobile, tablet, and desktop.
- **Clean Code**: Follows Django best practices and modular architecture.

## 🛠️ Tech Stack

- **Backend**: Django 5.1
- **Database**: SQLite3
- **Frontend**: Vanilla CSS (Modern), HTML5, Bootstrap 5.3
- **Icons**: Font Awesome 6

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or higher
- Pip (Python package manager)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/task-flow.git
   cd task-flow
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   # On Windows
   .\venv\Scripts\activate
   # On macOS/Linux
   source venv/bin/activate
   ```

3. **Install dependencies**:
   ```bash
   pip install django
   ```

4. **Run migrations**:
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   ```

5. **Start the server**:
   ```bash
   python manage.py runserver
   ```

6. **Access the app**:
   Open `http://127.0.0.1:8000` in your browser.
